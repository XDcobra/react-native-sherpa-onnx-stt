# SenseVoice Model

This directory should contain a **SenseVoice** model for offline speech recognition.

## Required Files

```
sherpa-onnx-sense-voice-zh-en-ja-ko-yue-int8/
├── model.onnx            (or model.int8.onnx for quantized)
└── tokens.txt            (REQUIRED)
```

## Model Type

- **Type**: `sense_voice`
- **Description**: Requires model.onnx and tokens.txt files

## Download

For download links and more information, see the [main README.md](../../../../../../README.md#supported-model-types).

Look for the **SenseVoice** model type in the "Supported Model Types" table.
