# WeNet CTC Model

This directory should contain a **WeNet CTC** model for offline speech recognition.

## Required Files

```
sherpa-onnx-wenetspeech-ctc-zh-en-cantonese/
├── model.onnx            (or model.int8.onnx for quantized)
└── tokens.txt            (REQUIRED)
```

## Model Type

- **Type**: `wenet_ctc`
- **Description**: Requires model.onnx and tokens.txt files

## Download

For download links and more information, see the [main README.md](../../../../../../README.md#supported-model-types).

Look for the **WeNet CTC** model type in the "Supported Model Types" table.
