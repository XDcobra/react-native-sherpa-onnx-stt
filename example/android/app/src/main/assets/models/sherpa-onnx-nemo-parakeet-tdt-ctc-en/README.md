# NeMo CTC Model

This directory should contain a **NeMo CTC** model for offline speech recognition.

## Required Files

```
sherpa-onnx-nemo-parakeet-tdt-ctc-en/
├── model.onnx            (or model.int8.onnx for quantized)
└── tokens.txt            (REQUIRED)
```

## Model Type

- **Type**: `nemo_ctc`
- **Description**: Requires model.onnx and tokens.txt files

## Download

For download links and more information, see the [main README.md](../../../../../../README.md#supported-model-types).

Look for the **NeMo CTC** model type in the "Supported Model Types" table.
