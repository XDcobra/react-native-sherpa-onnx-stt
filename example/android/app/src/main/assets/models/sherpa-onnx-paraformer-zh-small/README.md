# Paraformer Model

This directory should contain a **Paraformer** model for offline speech recognition.

## Required Files

```
sherpa-onnx-paraformer-zh-small/
├── model.onnx            (or model.int8.onnx for quantized)
└── tokens.txt            (REQUIRED)
```

## Model Type

- **Type**: `paraformer`
- **Description**: Requires model.onnx and tokens.txt files

## Download

For download links and more information, see the [main README.md](../../../../../../README.md#supported-model-types).

Look for the **Paraformer** model type in the "Supported Model Types" table.
